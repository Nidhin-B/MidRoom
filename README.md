# MidRoom
MidRoom — Minimalist Ambient Web Text Editor
A minimalist, immersive writing sanctuary designed for the void. Midroom provides a distraction-free, atmospheric darkroom for your thoughts, featuring a custom particle-driven environment and a persistent vault for your drafts. Built for creators who need a space that feels as deep as their stories.

Key Features:

Immersive Atmosphere: A fluid, gradient-based darkroom aesthetic with ambient, interactive spore particles.

Vault System: Secure, local-storage based management for all your drafts, with support for custom naming.

Mobile-First Design: Optimized for a seamless, tactile writing experience on handheld devices.

Distraction-Free: Clean, focused interface that keeps you locked in your creative flow.
